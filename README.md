# umamusume-model-replace
- replace umamusume models



# 使用方法

- 安装`Python3.8及以上版本`
- 安装依赖: `pip install -r requirements.txt`

```
UnityPy~=1.8.15
```

- 运行`main.py`

```shell
python main.py
```

- 根据提示操作即可。

```
[1] 更换头部模型
[2] 更换身体模型
[3] 更换尾巴模型(不建议)
[4] 更换头部与身体模型
[5] 修改角色身体贴图
[6] 更换抽卡开门人物
[7] 更换技能动画
[8] 更换G1胜利动作(实验性)
[9] Live服装解锁
[10] 清除Live所有模糊效果
[11] 修改角色头部贴图
[98] 复原所有修改
[99] 退出
```
